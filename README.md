# DICOM Exhibition Web

슈가폴에서 제공한, DICOM 이미지를 다루는 웹프로그램을 구현하는 과제입니다.

# 사용 기술
- **React.JS**, **TS**, **Vite**
- **cornerstone-core v2.6.1**, cornerstone-wado-image-loader
- **Tailwind CSS**, Shadcn/ui, prettier, eslint
